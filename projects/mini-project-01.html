Mini Project 01

Generalized Linear Models

For this project you and you team will be (1) evaluating a published article on research that incorporates generalized linear models (GLMs) in the analysis and (2) using the data from the article to replicate the analysis and explore new insights.

The goals of the project are to see how GLMs are used in research and to apply what you’ve learned to analyze real-world data using GLMs.

Workflow & due dates

  • Project Week 01 (Jan 20): Select the article and accompanying data set, and write the proposal.
  • Project Week 02 (week of Mon, Jan 24): Read and evaluate the article and submit plan for analysis. Peer review two team’s analysis plans.
  • Project Week 03 (week of Mon, Jan 31): Work on the analysis.
  • Project Week 04 (week of Mon, Feb 07): Finalize the analysis, presentation, and write up. Present in class on Wed, Feb 09.

Article + data set

The article for this project may be published in an academic journal or a reputable non-academic site. In addition, the article must follow the following criteria:

  • The analysis incorporates the use of at least one GLM that is not a linear regression model.
  • The original data set analyzed in the article or a comparable data set provided by the authors is accessible.

See the Tips below for tips on searching journal databases to find an article for the project.

Proposal

The proposal should include the following:

  • The citation for the article. No specific format is required.
  • Brief summary about why you chose the article.
  • Brief summary of the paper’s objective and primary research objective.
  • Name of the GLM(s) used in the paper and a short description of the response variable for each model.
  • A glimpse of the data set

You are only required to write the proposal for one article and dataset. Write the proposal in the proposal.Rmd, and push the .Rmd and knitted PDF to the GitHub repo by the due date.

Grading criteria

The proposal will be graded based on the following:

  • Data set is in the data folder (2 pts)
